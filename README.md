Ethics.Ai - Phase 1 MVP

AI Literacy: Risks & Responsible Use
Free, interactive ethics course for everyday people – no tech background needed. Learn AI risks, spot bias, practice scenarios, earn certificate & shareable badge.

🚀 Live Demo
View Live Demo (Deploy with one click!)

📱 Features (All Working!)
✅ 5 Interactive Modules (AI Basics → Spotting Bias)
✅ Keyword Flashcards + Diagrams + Real Examples
✅ 3 Practice Scenarios (Deepfakes, Bias, Academic Integrity)
✅ Final Quiz (5 questions, 80% pass → unlock certificate)
✅ Certificate Generator (PNG download + LinkedIn/WhatsApp/Instagram share)
✅ Progress Tracking (localStorage, no signup)
✅ Mobile-First Responsive (perfect on phone)
✅ Zero Backend (single HTML file, <200KB)
✅ Lighthouse 95+ (Performance, Accessibility, SEO)

Time to complete: 45-60 minutes
Target audience: Students, professionals, parents, small business owners

🎨 Design & Tech
Feature	Tech Used
UI/Animations	Vanilla CSS (Tailwind-inspired), GSAP-like fades
Fonts/Icons	Google Fonts (Plus Jakarta Sans), Font Awesome 6
Certificate	html2canvas (PNG export)
State	localStorage (progress persists)
Responsive	Mobile-first CSS Grid/Flexbox
Colors: Deep Blue (#1D4ED8), Purple (#7C3AED), Green (#047857)
No frameworks – pure HTML/CSS/JS for max speed
